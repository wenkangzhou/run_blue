## 四、图片加工建议

### 尺寸规格
- **轮播图**：推荐 1290×2796（iPhone 竖屏）或 2796×1290（横屏）
- **功能模块图**：推荐 1:1 或 4:3，宽度 800-1200px
- 实际可用浏览器 DevTools 设备模拟截图，再按比例裁切

### 加工工具
- **Figma**（推荐）：用 Frame 套截图，叠加文字层，导出高清 PNG
- **Canva**：有现成 App Store 截图模板
- **Keynote/PPT**：简单排版后导出

### 加工规范
1. **文字位置**：放在图片下半部分，避开主要 UI 内容
2. **文字样式**：
   - 标题：24-32px，白色或品牌色（像素风可用 `font-pixel`）
   - 副标题：16-18px，灰色
   - 底部叠加 `backdrop-blur` 毛玻璃效果底栏
3. **不要遮挡关键 UI**：AI 分析的心率条形图、热力图等核心数据要完整露出
4. **统一风格**：所有截图保持同一主题（推荐深色主题，与像素极简风格一致）

### 参考排版（Figma 示意）
